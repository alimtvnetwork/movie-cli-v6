# PHP Standards — Acceptance Criteria

**Version:** 1.0.0  
**Last Updated:** 2026-03-20

---

## AC-01: WordPress PHP

- [ ] Class naming follows WordPress PSR-4 autoloading conventions
- [ ] Database queries use $wpdb prepared statements exclusively
- [ ] Hook registration follows WordPress action/filter best practices

## AC-02: Code Quality

- [ ] Type declarations (parameter + return types) required on all functions
- [ ] Error handling uses WordPress WP_Error pattern consistently
- [ ] Input sanitization and output escaping follow WordPress security standards
